# seed-nunjucks

Basic project setup with templating engine.

##### Bower Components
* jQuery 2.1.4
* normalize-css 3.0.3

##### Build Stack
* Gulp
* SASS
* Nunjucks

## Requirements
* nodejs
* bower

## Setup

Install gulp and its plugins:
```
npm install
```

Install bower components:
```
bower install
```

Run gulp and start webserver with live-reload:
```
gulp
```
