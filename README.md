# seed-basic

Basic project setup based on HTML5 Boilerplate.

##### Node Modules
* jQuery 2.1.4
* normalize-css 3.0.3

##### Build Stack
* Gulp
* SASS

## Requirements
* nodejs

## Setup

Install gulp and its plugins:
```
npm install
```

Run gulp and start webserver with live-reload:
```
gulp
```
