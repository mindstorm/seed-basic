# seed-nunjucks

Basic project setup with templating engine.

##### Node Modules
* jQuery
* normalize-css

##### Build Stack
* gulp
* scss
* browser-sync
* jshint
* nunjucks

## Requirements
* nodejs

## Setup

Install gulp and its plugins:
```
npm install
```

Run gulp and start webserver with live-reload:
```
gulp
```
