# seed-angular-1

Basic project setup with AngularJS 1.x

##### Node Modules
* jQuery
* normalize-css
* AngularJS 1.x

##### Build Stack
* gulp
* scss
* browser-sync
* jshint

## Requirements
* nodejs

## Setup

Install gulp and its plugins:
```
npm install
```

Run gulp and start webserver with live-reload:
```
gulp
```
